# Instagram Images

## Jak přidat obrázky z Instagramu:

### 1. Otevři Instagram post v prohlížeči
Například: https://www.instagram.com/p/DQsCraPAhTW/

### 2. Udělej screenshot nebo stáhni obrázek
- **Screenshot:** Vyfoť jen obrázek (bez komentářů, lajků)
- **Nebo použij službu:** https://instadownloader.co/ 

### 3. Ulož obrázek do této složky
Pojmenuj je:
- `post-1.jpg`
- `post-2.jpg`
- `post-3.jpg`
- `post-4.jpg`

### 4. Commit a push
```bash
git add public/instagram/
git commit -m "Add Instagram images"
git push
```

## Doporučená velikost obrázků:
- **Minimálně:** 800x800px
- **Ideálně:** 1080x1080px
- **Formát:** JPG nebo PNG
- **Orientace:** Čtvercové (1:1)

## Rychlý tip:
Pro nejlepší kvalitu stáhni obrázky z Instagramu pomocí služby jako:
- https://instadownloader.co/
- https://downloadgram.org/
- https://igram.io/
